### Um repositório para armazenar minhas soluções para os exercícios do Java MOOC.
Criei esse repositório com o objetivo de conseguir acessar os exercícios pelo meu sistema windows ou linux.
Então se eu tiver resolvido alguns exercícios pelo windows, basta executar um `git pull` pelo linux e continuar os trabalhos por lá.

### O que é o Java MOOC?
Caso você tenha caído de paraquedas nesse repositório, o Java MOOC é um curso de introdução a programação e ao Java, disponibilizado (em inglês) gratuitamente pela Universidade de Helsinki. Caso lhe interesse ou ache que possa interessar alguém que você conheça, o link é esse:<br/>
[https://java-programming.mooc.fi](https://java-programming.mooc.fi)
